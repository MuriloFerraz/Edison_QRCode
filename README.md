# Edison_QRCode

Exemplo de controle de acendimento de LED utilizando
mensagem codificadas em QR Code

# Lista de materiais
- Intel Edison w Arduino Expansion Board
- Webcan (Logitech C270)
- Grove LED
- Grove-LCD RGB - I2C (Opcional)
- Cabo USB A-B
- Fonte 12V 1A
